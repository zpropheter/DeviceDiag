This tool is designed to help Mac Admins better understand results from a sysdiagnose. It is definitely a work in progress. Feel free to try it out.

# Main features:
- OS Update troubleshooting
- Declaration Reporting
- Status Item reporting
- Configuration Profile Analysis
- Basic device information ( PPPC, Managed Login Items, and Notifications )

# Installation
1. Download the main repo and contents
2. pip install -r /path/to/requirements.txt
3. /path/to/run.sh
4. open web browser and go to 127.0.0.1:5001
